# StudioLocation Backend

## 🎯 À propos du projet

**StudioLocation Backend** est une API REST développée en **Java** qui gère la location de studio.
Le projet est en cours de développement. L'api n'est pas encore déployée.

## Fonctionnement

Le projet suit une architecture en 3 couches (controller, service, model) avec les fonctionnalités :

### Fonctionnalités principales

- **Recherche dynamique** - En fonction des filtres, requis affichage des logements (avec pagination et tri)
- **Accès et sortie des données** - Mise place de dto avec validation
- **Erreurs** - Exception handling explicites
